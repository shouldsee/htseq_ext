### Overview

Python2 extension to HTSeq to perform transcript-oriented operations.


### Requires:

```
HTSeq
```

### Installation

#### User

```bash
pip2 install git+https://github.com/shouldsee/htseq_ext 
```

#### Dev

```bash
git clone git+https://github.com/shouldsee/htseq_ext && \
cd htseq_ext && \
python2 setup.py install
```
